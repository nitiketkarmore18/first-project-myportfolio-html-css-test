# 🚀 Personal Portfolio Website

A modern and fully responsive **developer portfolio website** built using the **MERN Stack ecosystem** to showcase projects, technical skills, certifications, and professional experience in an engaging and visually appealing way.

This project was originally developed during my Full Stack Web Development learning journey and demonstrates my understanding of modern frontend architecture, responsive UI design, reusable components, deployment workflows, and web performance optimization.
